Create an HOTP URI for use with Google Authenticator.

Note that this is not a URI for the QR code used by Google Authenticator. The URI produced by this method is used as the actual content of the QR code, and follows a special set of conventions understood by Google Authenticator, and other OTP apps.

Parameters

string $secret The shared secret.
string $label The label for the account.
string|null $issuer The issuer name.
integer|null $counter The current counter value.
integer|null $digits The number of password digits.
string|null $algorithm The algorithm to use.
boolean|null $issuerInLabel True if legacy issuer support should be enabled by prefixing the label with the issuer name.

Return Value

string The HOTP URI.
